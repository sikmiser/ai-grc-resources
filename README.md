# AI GRC Resources

Practical, curated resources for organizations implementing AI Governance, Risk & Compliance.

This repository collects regulatory summaries, cross-framework mappings, reusable templates, implementation guides, and working prototypes to help teams navigate AI regulations and build compliant workflows.

- Cross-framework comparison tables

### Prototypes

### Guides
